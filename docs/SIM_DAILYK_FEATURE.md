# 📈 模拟盘日 K 记录功能

**添加日期：** 2026-04-11  
**访问地址：** http://localhost:3001/sim/daily-k

---

## 🎯 功能说明

在模拟盘菜单下新增"日 K 记录"页面，提供标准的日 K 线图展示，包含：
- 📊 K 线图（红涨绿跌）
- 📈 成交量柱形图
- 📉 MACD 指标（DIF/DEA/MACD 柱）
- 🔍 股票选择和日期范围查询
- 🔭 数据缩放和拖拽

---

## 📊 页面功能

### 1. 查询条件

| 条件 | 说明 |
|------|------|
| **股票选择** | 从自选表中选择股票 |
| **日期范围** | 选择查询的日期范围 |
| **查询按钮** | 加载指定股票和日期的 K 线数据 |

### 2. K 线图表

**图表布局：**
```
┌─────────────────────────────────┐
│        K 线图（50% 高度）        │
├─────────────────────────────────┤
│     成交量柱形图（15% 高度）     │
├─────────────────────────────────┤
│   MACD 指标（15% 高度）         │
└─────────────────────────────────┘
```

**K 线图：**
- 红色：阳线（收盘价 ≥ 开盘价）
- 绿色：阴线（收盘价 < 开盘价）

**成交量：**
- 红色：上涨成交量
- 绿色：下跌成交量

**MACD 指标：**
- 黄线：DIF（快线）
- 青线：DEA（慢线）
- 红柱：MACD 正值（多头）
- 绿柱：MACD 负值（空头）

### 3. 交互功能

| 功能 | 操作 |
|------|------|
| **缩放** | 鼠标滚轮滚动 |
| **拖拽** | 鼠标左键拖动 |
| **底部滑块** | 拖动滑块调整显示范围 |
| **悬停提示** | 鼠标悬停查看详细数据 |

## 📝 使用示例

### 1. 选择股票

从下拉列表中选择要查看的股票：
- 600105.SH 永鼎股份
- 562500.SH 机器人 ETF
- ...

### 2. 选择日期范围

选择要查看的日期范围：
- 默认：最近 60 天
- 自定义：选择开始和结束日期

### 3. 查看 K 线图

- **K 线图** - 查看股价走势
- **成交量** - 查看交易量变化
- **MACD** - 查看趋势指标

### 4. 交互操作

- **缩放** - 鼠标滚轮向前/向后滚动
- **拖拽** - 鼠标左键按住拖动
- **滑块** - 拖动底部滑块调整显示范围
