## 🔍 问题描述

模拟盘概况页面显示的数据不正确：
- **显示值：** 总值 ¥200,000（初始值）
- **实际值：** 总值 ¥473,530.16（回测最新值）

**根本原因：**
- `sim_account` 表数据未自动同步
- `sim_daily_snapshots` 表有最新数据，但未更新到 `sim_account`

---

## ✅ 解决方案

### 1️⃣ 手动同步脚本

**文件：** `backend/sync_sim_account.py`

```bash
cd /home/lizr/pyProject/stockMonitor/backend
python3 sync_sim_account.py
```

### 2️⃣ API 自动同步

**修改：** `backend/app.py` - `get_sim_stats()` API

每次访问 `/api/sim-stats` 接口时自动同步最新数据。
