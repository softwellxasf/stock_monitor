## 🔧 技术指标计算

### MA 均线计算

```javascript
// 计算 MA 均线
const calculateMA = (data, period) => {
  const ma = []
  for (let i = 0; i < data.length; i++) {
    if (i < period - 1) {
      ma.push('-')  // 数据不足
      continue
    }
    let sum = 0
    for (let j = 0; j < period; j++) {
      sum += data[i - j][1]  // 使用收盘价
    }
    ma.push(sum / period)
  }
  return ma
}
```

### MACD 计算（预留）

```javascript
// 计算 MACD 指标
const calculateMACD = (data) => {
  // 1. EMA(12) - 12 日指数移动平均
  // 2. EMA(26) - 26 日指数移动平均
  // 3. DIF = EMA(12) - EMA(26)
  // 4. DEA = DIF 的 9 日 EMA
  // 5. MACD 柱 = 2 × (DIF - DEA)
  
  return { dif, dea, macd }
}
```

## 🌐 使用方式

### 访问地址

**URL:** http://localhost:3001/sim/daily-k

### 操作步骤

1. **选择股票**
   - 从下拉列表选择股票
   - 支持搜索过滤

2. **选择日期范围**
   - 默认：最近 60 天
   - 自定义：选择开始和结束日期

3. **查看 K 线图**
   - K 线图：红涨绿跌
   - 均线：MA5/MA10/MA20
   - 成交量：下方柱形图

4. **交互操作**
   - **缩放**：鼠标滚轮滚动
   - **拖拽**：鼠标左键拖动
   - **滑块**：底部滑块调整范围
   - **悬停**：查看 OHLC 详情

## 🎨 图表配置

### ECharts 配置要点

```javascript
const option = {
  backgroundColor: '#ffffff',
  animation: false,  // 禁用动画，提升性能
  
  // 高清渲染
  renderer: 'canvas',
  devicePixelRatio: window.devicePixelRatio || 1,
  
  // 双网格布局
  grid: [
    { height: '55%', top: '12%' },   // K 线图
    { height: '12%', top: '70%' }    // 成交量
  ],
  
  // 右侧 Y 轴
  yAxis: [
    { 
      position: 'right',
      name: '股价',
      axisLabel: { formatter: (v) => v.toFixed(2) }
    },
    { 
      gridIndex: 1,
      position: 'right',
      name: '成交量',
      axisLabel: { 
        formatter: (v) => {
          if (v >= 1000000) return (v/1000000).toFixed(1) + 'M'
          if (v >= 1000) return (v/1000).toFixed(0) + 'K'
          return v.toString()
        }
      }
    }
  ]
}
```
